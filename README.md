# 林教头风雪山神庙mud游戏设计
### 游戏背景：
林冲被高太尉发配到草料场，但是高太尉并不打算放过林冲，令人去火烧草料场构陷林冲，林冲机缘巧合下知道了高太尉的阴谋，将高太尉的手下杀了并开始了他的江湖生涯。
### 游戏内容：
游戏为分为剧情、养成、战斗三板块，在剧情中养成，在养成中战斗，让玩家能够体验到林冲风雪山神庙的故事情节。
在养成中的码头场景，添加了码头负责人智能体，由大模型随机生成码头场景任务内容。
### 游戏打开方式：
将MUD—VSS.exe和309764634.wav文件放在同一文件夹下，运行MUD—VSS.exe即可  
或构建CPP项目文件，将其中所有.h文件添加到库文件中，同时将所有.cpp文件添加到源文件中，并将修改main.cpp文件中的309764634.wav文件路径为本地路径，运行main.cpp文件即可。
